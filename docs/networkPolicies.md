`eck-operator`, like many other Big Bang packages, allows you to define additional network controls through Helm values.

This package uses the shared Big Bang `networkPolicies` values documented in the [Big Bang network policies guide](https://docs-bigbang.dso.mil/latest/docs/guides/using-bigbang/network-policies/).

See [`chart/values.yaml`](../chart/values.yaml) for the package defaults and available overrides.
